## PROGETTI

#### MyMoviesCollection - Applicazione Web FullStack per la Gestione di Collezioni Cinematografiche
*Un'applicazione web personale per catalogare e gestire la propria collezione di film, costruita con un'architettura moderna a microservizi.*
*   **Architettura:**
    *   Il frontend è un'interfaccia reattiva sviluppata in Angular e TypeScript, che sfrutta l'API di TMDB per la ricerca dei film.
    *   Il backend è un'API RESTful costruita in Java con Spring Boot per gestire la logica di business, la sicurezza e la persistenza dei dati.
    *   L'infrastruttura è containerizzata con Docker e il deploy è automatizzato tramite GitHub Actions.
*   **Funzionalità Principali:**
    *   Autenticazione utenti.
    *   Ricerca e inserimento di film da un database esterno (TMDB).
    *   Gestione di una "Collezione" personale e di una "Lista Desideri".
*   **Link Utili:**
    *   Sito Web: [my-movies-collection](https://tiziano-occhionero.github.io/my-movies-collection/)
    *   Repository Frontend: [tiziano-occhionero/my-movies-collection](https://github.com/tiziano-occhionero/my-movies-collection)
    *   Repository Backend: [tiziano-occhionero/mymovies-backend](https://github.com/tiziano-occhionero/mymovies-backend)
